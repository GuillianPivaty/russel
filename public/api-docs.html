⚓ Port de Russell — Documentation API

API REST privée — Gestion des catways et réservations

← Retour à l'accueil

🔒 Authentification requise

  • Toutes les routes (sauf POST /login et GET /) nécessitent une session active.
  • Se connecter via POST /login avec email et password.
  • Se déconnecter via GET /logout.

Authentification

POST/loginConnexion (email + password)
GET/logoutDéconnexion

Catways

GET/catways🔒 AuthLister tous les catways
GET/catways/:id🔒 AuthDétail d'un catway
POST/catways🔒 AuthCréer un catway
PUT/catways/:id🔒 AuthModifier l'état d'un catway
DELETE/catways/:id🔒 AuthSupprimer un catway

Réservations (sous-ressource des catways)

GET/catways/:id/reservations🔒 AuthLister les réservations
GET/catway/:id/reservations/:idReservation🔒 AuthDétail d'une réservation
POST/catways/:id/reservations🔒 AuthCréer une réservation
PUT/catways/:id/reservations/:idReservation🔒 AuthModifier une réservation
DELETE/catway/:id/reservations/:idReservation🔒 AuthSupprimer une réservation

Utilisateurs

GET/users/🔒 AuthLister les utilisateurs
GET/users/:email🔒 AuthDétail d'un utilisateur
POST/users/🔒 AuthCréer un utilisateur
PUT/users/:email🔒 AuthModifier un utilisateur
DELETE/users/:email🔒 AuthSupprimer un utilisateur
